# Next App Router with shadcn/ui Boilerplate

## Getting Started
### Prerequisites
- Node.js 18.0
- npm/yarn/pnpm

### Preinstalled
- Next.js 14
- shadcn/ui
- TailwindCSS
- lucid-react

### Usage
- Click "Use this template" > "Create a new repository".
- Set up your repository.
- Clone the created repository to your computer.
- Download it using a package manager
  - yarn
  
    ```
    yarn install
    ```
  - npm
  
    ```
    npm install
    ```
  - pnpm
  
    ```
    pnpm install
    ```
